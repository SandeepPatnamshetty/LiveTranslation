# How to Hard Reload Chrome Extension

## The error you're seeing is from the OLD code. Follow these steps:

### Option 1: Manual Reload (RECOMMENDED)

1. Go to `chrome://extensions/`
2. Find "Real-Time Video Translator"
3. Click **"Remove"** button (this won't delete your files)
4. Click **"Load unpacked"** again
5. Select the folder: `C:\SandeepKumarPatnamshetty\Projects\LiveTranslation`
6. Test again

### Option 2: Toggle Extension

1. Go to `chrome://extensions/`
2. Find "Real-Time Video Translator"
3. Toggle it **OFF** (disable)
4. Toggle it **ON** (enable)
5. Click the **reload icon** (🔄)
6. Test again

### Option 3: Restart Chrome

1. Close ALL Chrome windows
2. Reopen Chrome
3. Go to `chrome://extensions/`
4. Verify the extension is loaded
5. Test again

## After Reloading:

1. Open: https://www.youtube.com/watch?v=jNQXAC9IVRw

4. Open DevTools (F12) Console tab
