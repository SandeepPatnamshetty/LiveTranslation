# 🚀 Quick Start Guide

Get the Real-Time Video Translator up and running in **5 minutes**!

## ⚡ Prerequisites

- ✅ Google Chrome 88+ or Microsoft Edge
- ✅ OpenAI API key ([Get one here](https://platform.openai.com/api-keys))
- ✅ $5-10 in OpenAI credits

## 📦 Installation (2 minutes)

### 1. Download Extension

**Option A: Clone with Git**

```bash
git clone https://github.com/yourusername/video-translator-extension.git
cd video-translator-extension
```

**Option B: Download ZIP**

- Download from [GitHub](https://github.com/yourusername/video-translator-extension)
- Extract to a permanent folder

### 2. Add Icons (Required)

Create or download icons and place in `icons/` folder:

- `icon16.png` (16×16 pixels)
- `icon48.png` (48×48 pixels)
- `icon128.png` (128×128 pixels)

**Quick Fix**: See [icons/README.md](icons/README.md) for simple icon generation.

### 3. Load in Chrome

1. Open Chrome → `chrome://extensions/`
2. Enable **Developer mode** (top-right toggle)
3. Click **"Load unpacked"**
4. Select the `LiveTranslation` folder
5. Extension icon appears in toolbar! 🎉

## 🔑 Configuration (2 minutes)

### 1. Get OpenAI API Key

1. Visit [OpenAI API Keys](https://platform.openai.com/api-keys)
2. Sign in or create account
3. Click **"Create new secret key"**
4. Copy the key (starts with `sk-...`)
5. **Save it** - you won't see it again!

### 2. Add Credits (if new account)

1. Go to [Billing](https://platform.openai.com/account/billing)
2. Add payment method
3. Purchase $5-10 credits

### 3. Configure Extension

1. **Click extension icon** in Chrome toolbar
2. **Paste API key** in the field
3. Click **"Validate"** to test
4. Click **"Save"**
5. (Optional) Adjust settings:
   - Overlay position: Bottom Center (default)
   - Font size: Medium (default)
   - Check "Use GPT-4o Mini" for lower costs ✅

## 🎬 First Translation (1 minute)

### Test It Out

1. **Open YouTube**: https://www.youtube.com/watch?v=jNQXAC9IVRw
2. **Click extension icon**
3. **Click "Start Translation"**
4. **Play the video**
5. **Hover near bottom of video**
6. **See translations appear!** 🎉

### Tips for Best Results

- 🎯 Hover in the **bottom 20%** of the video
- 🔊 Ensure video has **audio** (not muted)
- 🌐 Works best with **clear speech** content
- ⚡ Translations appear with **1-2 second delay**

## 🎨 Customization

### Change Overlay Position

1. Click extension icon
2. Settings → Overlay Position
3. Choose: Bottom Center, Top Center, Bottom Left, or Bottom Right
4. Click "Save Settings"

### Adjust Font Size

1. Click extension icon
2. Settings → Font Size
3. Choose: Small, Medium, Large, or Extra Large
4. Click "Save Settings"

### Save Costs

1. Click extension icon
2. Check **"Use GPT-4o Mini"**
3. Click "Save Settings"
4. **Saves 60% on API costs!** 💰

## 🐛 Common Issues

### "Please set your API key"

➜ Click extension icon → Enter API key → Save

### "Invalid API key"

➜ Check key starts with `sk-` → Generate new key if needed

### "Quota exceeded"

➜ Add credits at [OpenAI Billing](https://platform.openai.com/account/billing)

### Overlay not showing

➜ Hover **near bottom of video** → Check extension is "Active"

### High latency (>3 sec)

➜ Enable "Use GPT-4o Mini" → Check internet speed
